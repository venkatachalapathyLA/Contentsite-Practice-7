AMAZING FEATURE

A theme that focus on ease of use and flexibility.
We have achieved this by adding all the necessary options where you expect to see them.

Improved Performance

Special attention has been given to reduce loading times. This is achieved by loading fewer files and by compressing their size. The result is an overall good performance on all devices.

Beautifully Responsive

Not just responsive... Mobile is really important and we know that. This was the main reason for hundreds of hours of work spent on improving mobile performance and user experience.

Extensive Option

Divided in 10 tabs the theme has all the necessary options to create various types of websites. Colors, backgrounds, fonts, sidebars, logos, header and footer can be edited very quickly.

One-Click Install

Need help starting to create pages with Visual Composer? Not a problem, because we have generated some samples for you. This can be done literally with a single click.

Spoken Vedio Tuts

Worried about the time that is needed to learn how to work with Fount? Need help with a specific feature? We got you covered with an extensive series of videos tutorials available on YouTube.

Font Awesome Iconss

The theme includes also the awarded Font Awesome Icon set currently featuring more than 400 icons. Add them to any page with an easy to use icon selector.

Easy To Uses

Tired of complicated themes or you're not that tech savvy? No worries, Fount was built with that in mind. We're certain that you will not need our help, but if you do, we'll be glad to assist you.

Visual Composer

Build all your content with ease and all the WordPress power at your fingertips. This page builder was tailored to fit the theme and you'll have almost 50 elements available to add to any page.

E-Commerce & Plugin Ready

Fully compatible with WooCommerce, with Fount you are ready to sell online. The theme was also optimized for other plugins such as MailChimp for WordPress, Revolution Slider & Contact Form 7.
